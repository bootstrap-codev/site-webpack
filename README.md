## Install dependencies

Run command:

```sh
yarn
```

Install the http-server package globally on your system:

```sh
npm install -g http-server
```

## Start development server

Run command:

```sh
yarn start
```

## Build

Run command:

```sh
yarn build
```

## Preview

Run command:

```sh
yarn preview
```
